# Image to PDF Converter

## Description

The Image to PDF Converter is a user-friendly tool designed to convert images into PDF documents effortlessly. This project supports various image formats and allows users to manage their images efficiently.

## Features

- **Multiple Format Support**: Convert images in different formats including JPG, PNG, BMP, and GIF into a single PDF document.
- **Batch Conversion**: Convert multiple images at once, saving time and effort.
- **Easy to Use**: Simple interface that makes the process easy for anyone, regardless of technical skill.
- **Customizable Settings**: Adjust the size and orientation of the output PDF.

## Usage

1. **Install the Required Dependencies**: Make sure you have any required libraries installed (e.g., PIL, FPDF).
2. **Run the Converter**:
   - Open your terminal.
   - Navigate to the project directory.
   - Use the command: `python converter.py input_folder_path output.pdf`
3. **View Your PDF**: Once the conversion is complete, check the output PDF in the specified location.

## Installation Instructions

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/pranavpk-05/pdf-converter.git
   cd pdf-converter
   ```

2. **Install Required Libraries**:
   Use pip to install any dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Converter**:
   Follow the usage instructions outlined above.
